Временная установка TWRP на оф. прошивку Xiaomi Redmi 3s /Note и пр.

July 4, 2017 · 2 min

Этот мануал можно считать уневерсальным, т.к. вероятнее всего он будет работать и на других смарфонах Xiaomi. Для всех манипуляций, которые требуеют наличия ПК, мной был использован ноутбук под управлением Debian 9 Stretch. Установка производилась на Xiaomi Redmi 3s. При желании мануал можно адаптировать  Мануал адаптирован и для ОС семейства Windows.

О том, как установить TWRP на постоянной основе читайте в этой статье.

Главным преимуществом данного метода является возможность пользоваться официальной прошивкой, обновляться по ОТА при этом имея Root на своем устройстве.

TWRP будет подгружен временно, после перезагрузки в тлф останется родное recovery.

Все основные и подготовительные действия аналогичны тем, которые описаны в статье Установка TWRP на глобальной прошивке Xiaomi Redmi 3/Redmi 3s/Redmi 3x. Отличается только пункт 5, где требуется пропустить две первых команды, воспользовавшись сразу командой “boot”.

Преждем чем переводить устройство в режим  Fastboot заранее копируем на флешку архив с superSu и прочие патчи.

Временная установка TWRP на оф. прошивку Xiaomi Redmi 3s#

cd ~/redmi3/twrp/
sudo fastboot boot twrp_redmi3sx.img

TWRP должен открыться в течении одной минуты.

ВАЖНО! После обновления по ОТА Root доступ будет удален, все дейсвия по установке TWRP на оф. прошивку придется выполнять еще раз.

Временная установка TWRP из под Windows#

Если у Вас еще не установлен MiFlash устанавливаем - скачать. Пользоваться им не будем, установка нужна только для инсталяции драйверов.

Важно. TWRP придется искать самим,  за исключением владельцев redmi3s\x. Если с поиском будут проблемы пишите в комментарии. 

Для избежания возможных ошибок метод упрощен.

  1. Скачаем софт, необходимый для работы со смартфоном— скачать minimal_adb_fastboot;
  2. Установим adb по адресу — C:/adb (пусть может быть любым, но так проще);
  3. Распакуем содержимое архива с TWRP по адресу— C:/adb ;
  4. Переименуем файл с расширением .img (например  TWRP_Ido.img) в TWRP.img ;
  5. Скачаем этот батник, после чего скопируем его в C:/adb ;
  6. Выключим устройство;
  7. Переводим смартфон в режим fastboot — для этого нужно зажмать кнопку включения и громкость минус, после вибрации отпустить кнопку включения.
  8. На экране должен появиться заяц в шапке;
  9. В папке с файлами ищем — boot_twrp.bat;
  10. Запускаем его (желательно сделать это от имени администратора, во избежание проблем).
  11. В случае успешной установки TWRP запустится сам.
