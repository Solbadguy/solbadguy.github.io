Установка TWRP на глобальной прошивке Xiaomi Redmi 3/Redmi 3s/Redmi 3x

October 20, 2016 · 2 min

 

Данная инструкция к действию пригодится в первую очередь тем, кто по каким-либо причинам не хочет пользоваться кастомными прошивками от известных ромоделов (Xiaomi.eu, multirom и прочих). Данный способ позволит установить TWRP на разблокированный загрузчик телефона, а так же получить рут-права.

В статье будет рассмотрена Установка TWRP на глобальной прошивке Xiaomi Redmi 3/Redmi 3s/Redmi 3x используя ПК под управлением Ubuntu и Windows. Все действия аналогичны и для других устройств, мануал можно считать универсальным, за исключением архивов с TWRP, которые подойдут только для Redmi 3 /Redmi 3s/ Redmi 3x.

Все дальнейшие действия подразумевают, что вы выполнили следующее:

  1. Разблокировали bootloader официальным способом (MiFlashUnlock);
  2. Скачали архив с TWRP -
  • скачать TWRP для Redmi 3
  • скачать TWRP для Redmi 3s
  • скачать TWRP для Redmi 3x

Установка TWRP на глобальной прошивке Xiaomi Redmi 3/Redmi 3s/Redmi 3x#

ОС Windows#

Установка:

  1. Скачаем софт, необходимый для работы со смартфоном - скачать minimal_adb_fastboot;
  2. Установим adb по адресу - C:/adb (пусть может быть любым, но так проще);
  3. Распакуем содержимое архива с TWRP по адресу - C:/adb ;
  4. Выключим устройство;
  5. Переводим смартфон в режим fastboot - для этого нужно зажмать кнопку включения и громкость минус, после вибрации отпустить кнопку включения.
  6. На экране должен появиться заяц в шапке;
  7. В папке с распакованными файлами ищем файл - flash_twrp_redmi3sx.bat (flash_twrp_ido_protect.bat для Redmi3);
  8. Запускаем его (желательно сделать это от имени администратора, во избежание проблем).
  9. В случае успешной установки TWRP запустится сам.

ОС Ubuntu (Linux)#

Установка:

1. Установим необходимые утилиты:

sudo apt-get install android-tools-adb android-tools-fastboot

2. Переводим выключенное устройство в режим fastboot;

3. Проверим видит ли ПК ваш смартфон:

sudo fastboot devices

В результате терминал должен будет показать что-то подобное (это значит что убунту видит смартфон):

1904bd21 fastboot

4. Переходим в папку с TWRP:

cd ~/redmi3/twrp/

5. Поочередно вводим следующие команды:

sudo fastboot erase recovery

sudo fastboot flash recovery twrp_redmi3sx.img

sudo fastboot boot twrp_redmi3sx.img

В случае успешной установки терминал ответит “ОКAY” на все введенные команды.  TWRP должен открыться сам.

Для получения рут-прав достаточно скачать архив с superSU и установить его через TWRP.
