Прошивка сматфонов Xiaomi в Ubuntu

October 17, 2016 · 2 min

Среди владельцев сматфонов Xiaomi и пользователей *buntu, есть много тех, кто не знает о том, что их девайс можно прошить прямо из консоли, ни какие программы на вроде Miflash при этом не понадобятся. Автор статьи, описывая действия, предварительно скачал архив с прошивкой рассчитанной для установки из MiFlash. Материал в первую очередь написан для тех, кто не работает под Windows.

Ранее я уже писал о том, как проверить статус блокировки загрузчика смартфона в Ubuntu/Debian, на сей раз пойдем дальше. Я надеюсь, что все важные файлы, которые были на твоем смартфоне уже сохранены на внешней карте/ПК/облаке.

Прошивка сматфонов Xiaomi в Ubuntu#

1. Установим необходимые утилиты:

sudo apt-get install android-tools-adb android-tools-fastboot

2. Переводим выключенное устройство в режим fastboot: 

зажимаем громкость минус и питание.

3. Проверим видит ли ПК ваш смартфон:

sudo fastboot devices

В результате терминал должен будет показать что-то подобное (это значит что убунту видит смартфон):

1904bd21 fastboot

4. Переходим в папку с прошивкой:

cd ~/redmi3/update/V8.0.5

5. Начинаем! - прошивка сматфонов Xiaomi в Ubuntu

sudo sh ./flash_all.sh

Запустив flash_all важно понимать, что вся информация из памяти устройства будет удалена.

Теперь осталось дождаться окончания процесса прошивки смартфона. В случае, если он сам не перезагрузится используем следующую команду:

sudo fastboot reboot

К слову о скриптах, в архиве он не один, напишу о каждом:

  • flash_all_except_data_storage.sh  - не стирать /data и внешнюю карту
  • flash_all_except_storage.sh - прошить но не трогать внешнюю карту
  • flash_all.sh - удалить все и прошить (данные на внешней карте сохраняются).
  • flash_all_lock.sh - судя по всему может заблокировать бутлоадер назад.

Для “голой” прошивки рекомендуется использовать скрипт “flash_all.sh”. Загрузка может занять много времени, стоит набраться терпения.
