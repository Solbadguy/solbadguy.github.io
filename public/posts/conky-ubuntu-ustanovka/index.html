Conky Ubuntu - установка

June 15, 2015 · 1 min

Наверняка многие пользователи ПК когда-нибудь задавались целью получения удобного доступа к системной информации своего компьютера. Это особенно актуально для владельцев портативной техники (ноутбуки), т.к. те зачастую могут сильно перегреваться. 

Для пользователей windows и linux-подобных систем существует множество программ, которые в том или ином виде представляют нужную нам информацию, однако в плане гибкости и возможности настройки (под себя) они не могут тягаться с Conky. Изначально программа была реализована только под Linux-подобные системы, однако сейчас она доступна на всех платформах.

Установка Conky#

sudo apt-get install conky

Т.к. наша цель получение системной информации, давайте установим несколько утилит, которые позволят нам видеть информацию с датчиков компьютера:

sudo apt-get install curl lm-sensors hddtemp

Отмечу, что из коробки Ubuntu не может отображать температуру элементов, для этого требуется установка дополнительных пакетов, которые устанавливаются командой выше.

Для получения информации с датчиков в консоли делаем следующее:

sensors

 Запуск и остановка Conky#

Для запуска используем:

conky -d

Для остановки:

pkill conky

Для добавления conky в автозагрузку можно использовать стандартный пункт меню в настройках, который позволяет добавлять приложения в автозагрузку.

Конфигурационный файл Конки находится в домашней директории (.conkyrc), надо помнить, что он является скрытым.
