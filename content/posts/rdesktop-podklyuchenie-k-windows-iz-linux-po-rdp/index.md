Простой способ подключения к станциям под управлением Windows из под любого Linux-based дистрибутива.<!--more-->

### **Rdesktop установка**

Установка простая.

```
apt-get install rdesktop    #Команду стоит вводить от лица root
```

* * *

### **Подключение к серверу**

```
rdesktop -z -P -g 1280x900 -u USERNAME -p PASSWORD SERVER_IP
```

где:

- \-z включает сжатие потока
- \-P кэширование
- \-g задает геометрию окна
- \-u имя пользователя на сервере
- \-p пароль.

Опцию "-g" можно заменить опцией "-f". Тогда  rdesktop будет открыт на весь экран. Выход  можно будет совершить нажатием клавиш Ctrl-Alt-Enter.

* * *

### Быстрый запуск

В случае если постоянно приходится подключаться к одной и той же машине, то можно создать простой скрипт:

```
#!/bin/bash
rdesktop -z -P -g 1280x900 -u USERNAME -p PASSWORD SERVER_IP
```

После чего требуется сделать его исполняемым

```
chmod +x name_script
```
