У некоторых убунтуводов во время установки дистрибутива основанного на kubuntu plasma 5 с флешки возникают проблемы.<!--more-->

> ```
> Can't open  /lib/init/init-d-script saned disabled; edit /etc/default/saned 
> 
> ```

именно такое сообщение видят пользователи во время установки. На этом этапе как правило установка стопорится и последующая загрузка системы невозможна.

```
Однако расстраиваться и перезаписывать образ не нужно - выход есть. 

Когда ошибка появляется, нажмите Ctrl + Alt + F1 и выполняйте следующую команду:
```

> sudo killall Xorg

Затем нажмите Ctrl + Alt + F7(возвращение к графическому режиму)/ В моем случае это было не обязательно - загрузка произошла сама.
