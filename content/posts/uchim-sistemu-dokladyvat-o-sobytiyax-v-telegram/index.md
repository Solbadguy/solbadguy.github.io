При написании скриптов, мне иногда нужнен отчет об их работе. Самый простой вариант это E-mail. Однако имея на своем смартфоне несколько мессенджеров, неплохо было бы получать уведомления туда. <!--more-->

### Учим систему докладывать о событиях в Телеграм

- [Создаем Бота](https://core.telegram.org/bots#6-botfather)
- Получаем его ID через [@BotFather](https://telegram.me/BotFather)
- Получаем ID чата используя команду:

```
curl https://api.telegram.org/bot<Token>/getUpdates
```

_в случае если ничего не пришло в ответ - отправьте в чат любое сообщение от своего имени._

- Составляем команду для отправки уведомления в чат с ботом.

```
curl "https://api.telegram.org/bot<Token>/sendMessage" -d 'chat_id=<ChatID>&text=У Вас получилось!'
```

 

### UPD.

_На первых порах Curl не отрабатывал, ссыллся на bad gateway. Ответ очень прост - блокировки провайдера._
