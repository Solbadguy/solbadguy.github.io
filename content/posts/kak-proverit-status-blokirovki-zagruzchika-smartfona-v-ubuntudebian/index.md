Как вы знаете с недавнего времени смартфоны от Xiaomi продаются с заблокированным загрузчиком, о причинах говорить не буду, лучше давайте узнаем как проверить статус бутлоадера в Linux-подобных ОС.<!--more--> 1. **Накатим adb**

```
sudo apt-get install android-tools-adb android-tools-fastboot
```

2\. **Переводим выключенное устройство в режим fastboot** (зажимаем громкость минус и питание) 3. **Проверим видит ли ПК ваш тлф**

```
sudo fastboot devices
```

В результате должно появиться цифровое значение - это ваше устройство. 4. **Выведем информацию о тлф**

```
sudo fastboot oem device-info
```

Среди всех результатов выполнения, есть и статус бутлоадера, если получили сообщение _< waiting for device >_ - значит команда дана не от администратора.

Ищем строку:

**Device unlocked**: **false** (заблокирован или неразблокирован) или **true** (разблокирован).
