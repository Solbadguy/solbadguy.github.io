Зачастую вебмастера могут сталкиваться с ограничениями хостинга на размер загружаемой базы. Для решения этой проблемы нужно писать в саппорт, ждать ждать ждать.... Но ведь есть и другие способы.<!--more-->

## Импорт базы msql прямо из консоли

Для начала хочу отметить, что далеко не все хостинг-провайдеры предоставляют доступ к консоли.

Подключение осуществляется либо через средства в панели управления, либо через штатные средства - [putty](http://www.chiark.greenend.org.uk/~sgtatham/putty/download.html).

Для загрузки базы на сервер используем scp (либо пользуемся простым FTP:

```
scp data.sql ivan@example.server.ru:/home/ivan
```

 _Файл в моем примере загрузится в директорию /home/ivan_

Логинимся на сервере.

Импортируем базу:

```
mysql -u username -p -h localhost DATA-BASE-NAME < data.sql
```

Базу данных я создал предварительно, поэтому в примере нужно вписать ее имя.
