Наверное у многих были ситуации, когда под руками только консоль, а время узнать нужно.<!--more-->

Для того, чтобы посмотреть текущее время через консоль Linux достаточно ввести следующую команду

```
date
```

Для того, чтобы поменять время консоль Linux нужно ввести следующую команду:

```
date MMDDhhmmCCYY.ss
```

где, MM — является месяцем, DD — днем, hh — часом, mm — минутой, CCYY — годом (вводим так - хххх), ss — секундами. Действие нужно выполнять из под рута.

Для того, чтобы посмотреть полный список команд, которые помогут вам изменить/настроить время через консоль Linux достаточно ввести

```
date --help
```
