Даже самый удачный "минимализм" лучше настраивать "под себя", давайте разберемся чем может помочь Elementary tweaks и как его установить. <!--more-->

Elementary tweaks - это утилита, разработанная для тонкой настройки внешнего вида Elementary OS.

1. В список основных возможностей утилиты можно входит:
2. Изменение иконок, курсора,  темы (GTK);
3. Изменение шрифтов;
4. Настройка анимации;
5. Настройка теней;
6. Настройка док-бара;
7. Настройка встроенного ФМ, возможность включения/выключения функции открытия папок и файлов одним щелчком.

## Установка Elementary tweaks:

```
sudo add-apt-repository ppa:mpstark/elementary-tweaks-daily
sudo apt-get update
sudo apt-get install elementary-tweaks
```

Проинсталировав нужный пакет, можно открывать настройки системы, если установка прошла корректно, то вы увидите новый пункт.
