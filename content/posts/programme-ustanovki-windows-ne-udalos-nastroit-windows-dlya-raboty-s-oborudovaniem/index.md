Нужно было установить Windows 7 на китайский ноутбук, при установке с флешки постоянно возникала ошибка "Программе установки Windows не удалось настроить Windows для работы с оборудованием". Что интересно, выскакивала ровно тогда, когда процесс настройки оборудования заканчивался на 100%.<!--more-->

### Решение ошибки

* * *

Это заняло какое-то время, а решение как чаще всего бывает было простым.

После того как выскочила ошибка выполним:

- Shift+F10 (на ноутбуке может потребоваться добавить в комбинацию клавишу Fn)
- Открылась командная строка
- C:\\Windows\\System32\\oobe\\msoobe.exe
- Открылось окно с предложением ввести имя вашей учетной записи, продолжаем установку.

* * *

После выполнения вышеуказанных действий процесс инсталяции удачно завершился, теперь ноутбук спокойно работает под управлением Windows 7.
