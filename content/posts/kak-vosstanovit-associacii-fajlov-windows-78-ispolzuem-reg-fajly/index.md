Бывает такое, что в результате небрежных/необдуманных действий пользователя (реже ошибок в самой системе) в ОС Windows 7/8 могут нарушиться стандартные ассоциации файлов и приложений, которые должны их открывать. В результате ярлыки приложений могут открываться через Word, а архивы через системный плеер.<!--more-->

В интернете можно найти огромное количество программ, которые помогают исправить эту ошибку, однако это не наш путь.

## Как восстановить ассоциации файлов Windows 7/8?

Для восстановления ассоциаций я предлагаю отредактировать их значения в реестре ОС Windows. Это обезопасит Вас от скачивания и установки сомнительных программ.

Ребята с сайта [sevenforums.com](http://www.sevenforums.com/tutorials/19449-default-file-type-associations-restore.html) заранее об этом позаботились и выложили набор готовых .reg файлов, которые помогут восстановить ассоциации файлов Windows 7/8.

Скачать по файлы по одному можно на их сайте.

Скачать все сразу можно кликнув по [ссылке](http://admin.netlab-kursk.ru/upload/default_extentions.zip).

### Дальнейшие действия по восстановлению ассоциаций файлов Windows 7/8:

- Распаковываем архив
- Выбираем нужное расширение
- Запускаем .reg файл
- Соглашаемся с предупреждениями системы
