Зачастую, при просмотре состояния открытых соединений через командную строку, в поле статус отображаются разные значения. Давайте разберемся в том, о чем они сообщают.<!--more-->

| ESTABLISHED | Соединение установлено |
| --- | --- |
| SYN\_SENT | Попытка установить соединение |
| SYN\_RECV | Был получен запрос на соединение |
| FIN\_WAIT1 | Порт закрыт, соединение завершается |
| FIN\_WAIT2 | Соединение закрыто, в ближайшее время процесс будет завершен |
| TIME\_WAIT | Режим ожидания |
| CLOSE | Соединение закрыто |
| CLOSE\_WAIT | Соединение закрыто, завершение соединения |
| LISTEN | Программа слушает данный порт, на наличие входящих соединений |
| CLOSING | Соединение через этот порт закрыто, однако приложения не отправили все данные |
| UNKNOWN | Состояние соединения неизвестно |
